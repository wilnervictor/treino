PLANILHA DE TREINO

PPL - Push, Pull, Legs

Treino A - Push
(Peito, Ombros, Tríceps)
»Supino reto com barra: 4 séries x 8-10
»Supino inclinado com halteres: 3 séries x 10-12
»Elevação lateral com halteres: 3 séries x 10-12
»Desenvolvimento de ombros na máquina: 3 séries x 8-10
»Tríceps corda na polia: 3 séries x 10-12
»Tríceps testa com unilateral: 2 séries x 10-12
Total séries por grupo: Peito(7) Ombros(6) Tríceps(5)
Treino B - Pull
(Costas, Bíceps)
»Remada curvada com barra: 4 séries x 8-10
»Puxada frontal pegada aberta: 3 séries x 8-10
»Remada baixa no cabo: 3 séries x 10-12
»Rosca direta com barra reta: 3 séries x 8-10
»Rosca martelo com halteres: 3 séries x 10-12
Total séries por grupo: Costas(10) Bíceps(6)
Treino C - Legs
(Pernas)
»Agachamento livre: 4 séries x 8-10
»Leg press: 3 séries x 10-12
»Flexora deitada: 3 séries x 10-12
»Elevação de panturrilha em pé: 4 séries x 15-20
»Abdômen: 3 séries x 10-12
Total séries: Quadríceps(7) Posteriores(3) Panturrilha(4)
Dicas gerais
  • Progressão: Aumente o peso ou repetições a cada 1-2 semanas, mantendo 6-12 repetições por série.
  • Nutrição: Superávit calórico (300-500 kcal acima da manuteção, 2680 kcal/dia.) 100-150g de proteína.
  • Recuperação: Durma 7-9 horas por noite. Se sentir fadiga com 6 dias, reduza 5.
  • Aquecimento: 5-10 minutos de mobilidade + 1-2 séries leves por exercício.
